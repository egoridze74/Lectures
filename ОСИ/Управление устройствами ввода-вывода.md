#оси 
## Классы устройств
- байт-ориентированные - инфа передаётся байтами. Подходит для более быстрых устройств (мышка, клава)
- блок-ориентированные - инфа передаётся блоками. Подходит для более медленных устройств (принтер)

## Устройство
Каждое устройство можно представить в виде двух компонент:
- механическая компонента - аппаратная часть устройства ввода-вывода ("железо")
- электронная компонента (контроллер) - печатная плата, которая сопрягается с шиной компьютера через разъём. Служит для преобразования выходов устройства в сигналы, воспринимаемые компьютером
