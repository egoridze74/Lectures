#оси 
## Классы устройств
- байт-ориентированные - инфа передаётся байтами. Подходит для более быстрых устройств (мышка, клава)
- блок-ориентированные - инфа передаётся блоками. Подходит для более медленных устройств (принтер)

## Устройство
Каждое устройство можно представить в виде двух компонент:
- механическая компонента - аппаратная часть устройства ввода-вывода ("железо")
- электронная компонента (контроллер) - печатная плата, которая сопрягается с шиной компьютера через разъём. Служит для преобразования выходов устройства в сигналы, воспринимаемые компьютером

## Контроллер
**Контроллер** - регистр ввода-вывода, который содержит дополнительную опцию - **прерывание**, при помощи которого контроллер может сообщить процессору, что его регистры готовы для записи/чтения.

**IRQ** (interrupt Request Line) - линия запроса прерывания является одним из входов для отправки сигнала процессору.

**Программное обеспечение** представляет собой многослойный пакет, разрабатываемый из принципа единообразных именований и независимости устройств.

## Уровни обработки ввода-вывода
1. Обработчики прерываний
2. Драйверы устройств ввода-вывода
3. Независимые от аппаратуры программные модули
4. Прикладные программы
