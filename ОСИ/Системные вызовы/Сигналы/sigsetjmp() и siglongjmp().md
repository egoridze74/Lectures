#оси #системные_вызовы
При помощи sigsetjmp() и siglongjmp() можно прыгнуть в другое место в программе (как goto, но для сигналов).
- sigsetjmp() - ставим метку, куда прыгать
- siglongjmp() - прыгаем на метку

Пример:
```
sigsetjmp();
printf("Введите имя файла:");
read(0, ...)
```
А в обработчик [[sigaction]] ставим siglongjmp()

### Синтаксис этих системных вызовов
```
int sigsetjmp(sigjmp_buf stack, int m);
```
- sigjmp_buf - содержит все состояния текущего процесса до метки sigsetjmp()
- m  - флаг, который говорит, сохранять ли нам сигнальную маску процессора (0 - не сохранять, иначе - сохранять)

```
int siglongjmp(sigjmp_buf stack, int val);
```
- sigjmp_buf - содержит все состояния текущего процесса до метки sigsetjmp()
- val - возвращаемое значение последней выполненной функции sigsetjmp
