## Вычисление арифметических выражений
`expr <арифм_выражение>` или `let <арифм_выражение>`
- Возможные операции внутри expr: $+, -, *, /, \%$
- Возможные операции внутри let: $+, -, *, /, \%, >, <, \geq, \leq, =, !=, \&, |, !$
Пример:
```
a = 2
a = `expr $a + 7`
b = `expr $a / 3`
c = `expr $a '*' $b`

echo a=$a $b c=$c
```

Сохранили это всё в файл под именем lab. теперь надо сделать его выполнимым:
`chmod ugo + x` - дали права и пользователю, и группе, и остальным на исполнение

Пример с символами:
```
x = abcde
w = `expr "$x" : 'abc'`
y = `expr "$x" : 'abd'`
z = `expr "$x" : ".*"`

echo $w $y $z
```

Пример с циклом:
```
a = 1
while let "a <= 3"
	do
		echo $a Hello
		let "a =a+1"
	done
	
Вывод:
1 Hello
2 Hello
3 Hello
```

## Проверка файлов, числовых величин и цепочек символов
`test выражение [и/или ...]`

### Ключи для файлов
- `-p` - проверка существования файла и является ли он каналом
- `-l` - проверка существования файла и является ли он ссылкой
- `-e` - проверка существования файла и является ли он файлом
- `-s` - имеет ли файл ненулевой размер
- `-r` - проверка на чтение
- `-w` - проверка на запись
- `-x` - проверка на выполнение
Пример: 
`test -s a.txt`
`echo $?` - если существует, то вернет 0, иначе 1

`test` ещё можно заменить на `[]`:
`[ -s a.txt] = test -s a.txt`

### Ключи для сравнения чисел
`test число отношение число`
- `-lt` это <
- `-le` это <=
- `-gt` это >
- `-ge` это >=
- `-eq` это ==
- `ne` это !=
Пример:
`[ $a -eq 5 ]`
`echo $?`

### Ключи для сравнения цепочек символов
- `test -n цепочка` - проверка существования цепочки символов
- `test -z цепочка` - проверка на отсутствие цепочки символов
- `test цепочка=(!=)цепочка` - проверка равенства/неравенства цепочек символов
Пример:
`a = dasha`
`test -n $a` вернет 0
`test -n $b` вернет 1

- "-o" - логическое ИЛИ
- "-a" - логическое И
- "!" - отрицание

## Конструкции if - сравнение
```
if - условие
then - список команд 1
else - список команд 2
fi
```

```
Пример:
if test $a -gt 5
then echo привет
else echo пока
```

## Конструкции case
```
case выражение in
	модель_1) список_команд_1;;
	модель_2) список_команд_2;;
	...
	модель_n) список_команд_n;;
esac
```

Пример:
```
a = `ls -l a.txt | cut -b 1-1` - вырезать один символ
ls -l a.txt
-rw-rwxr-- a.txt

case $a in
	'-') echo - файл обычный;;
	'd') echo - файл каталог;;
	...
	'l') echo - файл символ сс;;
esac
```

## Циклы
### while/until
```
while/until выражение
do
	список_команд
done
```

Пример:
```
x = 0
until [ $x -eq 10 ]
do
	echo Dasha
	x='expr $x + 1'
done

chmod u + x lab или sh lab
./lab
```

### for
```
for переменная [in список]
do
	список_команд
done
```

Пример:
```
for i in 1 35 sasha 10
do
	echo Hello $i
done

Выведет:
Hello 1
Hello 35
Hello sasha
Hello 10
```

Или другой:
```
n = 0
for i
do
	n = ' expr $n + 1'
	echo Argument $n : $i
done

Теперь запускаем:
./lab oleg 2 nina

Argument 1: oleg
Argument 2: 2
Argument 3: nina
```

## trap
trap позволяет прописать инструкции для реакции на сигналы
```
trap "
	команда 1
	команда 2
	...
	команда N
"
```

Пример:
```
trap "prer" 2
i = 0
prer()
{
	echo количество итераций $i
	exit1
}

while:
do
	i='expr $i + 1'
	echo $i
	sleep 1
done
```
