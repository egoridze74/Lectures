## Вычисление арифметических выражений
`expr <арифм_выражение>` или `let <арифм_выражение>`
- Возможные операции внутри expr: $+, -, *, /, \%$
- Возможные операции внутри let: $+, -, *, /, \%, >, <, \geq, \leq, =, !=, \&, |, !$
Пример:
```
a = 2
a = `expr $a + 7`
b = `expr $a / 3`
c = `expr $a '*' $b`

echo a=$a $b c=$c
```

Сохранили это всё в файл под именем lab. теперь надо сделать его выполнимым:
`chmod ugo + x` - дали права и пользователю, и группе, и остальным на исполнение

Пример с символами:
```
x = abcde
w = `expr "$x" : 'abc'`
y = `expr "$x" : 'abd'`
z = `expr "$x" : ".*"`

echo $w $y $z
```

Пример с циклом:
```
a = 1
while let "a <= 3"
	do
		echo $a Hello
		let "a =a+1"
	done
	
Вывод:
1 Hello
2 Hello
3 Hello
```
