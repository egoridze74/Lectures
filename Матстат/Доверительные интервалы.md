## Доверительный интервал для параметра $\theta$ 
У нас есть пара статистик $T_1(\vec{x}), \ T_2(\vec{x})$ такие, что:
$$P(T_1(\vec{x}) \leq \theta \leq T_2(\vec{x})) \geq \gamma$$

## Центральная статистика
$G(\vec{x}, \theta)$ - **центральная статистика**, если:
1) $G(\vec{x}, \theta)$ - монотонна по $\theta$ при фиксированной выборке
2) Распределение $G(\vec{x}, \theta)$ не зависит от $\theta$

Находим парочку квантилей, между которыми: 
$P(\gamma_1 \leq G(\vec{x}, \theta) \leq \gamma_2) = \alpha$
Берём обратную для $G(\vec{x}, \theta)$:
$P(G^{-1}(\vec{x}, \gamma_1) \leq \theta \leq G^{-1}(\vec{x}, \gamma_2)) = \alpha$

### Пример
$R[0, \theta]$
$x_1, \dots, x_n$
$x_{(n)}: \ F_{x_{(n)}}(t) = (\frac{t}{\theta})^n, \ 0 \leq t \leq \theta$
$\theta x_{(n)}$ - центральная статистика?

$P(\theta x_{(n)} \leq t) = P(x_{(n)} \leq \frac{t}{\theta}) = \frac{t^n}{\theta^{2n}}$
Тогда $\theta x_{(n)}$ - не центральная статистика

А $\frac{x_{(n)}}{\theta}$ - центральная статистика?
$\frac{x_{(n)}}{\theta}: \ P(\frac{x_{(n)}}{\theta} \leq t) = P(x_{(n)} \leq \theta t) = t^n$

Да, всё норм. Тогда попробуем подобрать квантили:
$P(\gamma_1 \leq \frac{x_{(n)}}{\theta} \leq \gamma_2) = 0.9$
$\begin{cases} \gamma_2^n - \gamma_1^n = 0.9 \\ min(\gamma_2 - \gamma_1) \end{cases}$
$P(\frac{x_{(n)}}{\gamma_2} \leq \theta \leq \frac{x_{(n)}}{\gamma_1}) = \alpha$
