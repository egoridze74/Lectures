[[Доверительные интервалы]]

## Выборочные моменты
$\overline{x} = \frac{1}{n} \sum x_i, \ E x_i^2 < \infty$
$P(\gamma_1 \leq \frac{\overline{x} - Ex_1}{\sqrt{\frac{Dx_1}{n}}} \leq \gamma_2) = P(\gamma_1 \leq \sqrt{n} - \frac{\overline{x} - Ex_1}{\sqrt{Dx_1}} \leq \gamma_2)$, где $G(\vec{x}, \theta) = \overline{x} - Ex_1$
$P(\gamma_1 \leq \sqrt{n} - \frac{\overline{x} - Ex_1}{\sqrt{Dx_1}} \leq \gamma_2) = P(\frac{\gamma_1}{\sqrt{n}} \leq G(\vec{x}, \theta) \leq \frac{\gamma_2}{\sqrt{n}}) \approx Ф(\gamma_2) - Ф(\gamma_1)$, где $Ф(x) = \int\limits_{- \infty}^{x} \frac{e^{-\frac{t^2}{2}}}{\sqrt{2 \pi}}dt$

### Пример
$Bi(m, \theta)$
$\overline{x} = \frac{1}{n} \sum\limits_{i = 1}^{n} x_i$
$Ex_1 = m \theta, \ Dx_1 = m \theta (1 - \theta)$
$\gamma = 3.26$
$\gamma_1 = -\gamma, \ \gamma_2 = \gamma$
$(-\gamma \leq \sqrt{n} \frac{\overline{x} - m\theta}{\sqrt{m \theta (1 - \theta)}} \leq \gamma) = Ф(\gamma) - Ф(-\gamma) = 2Ф(\gamma) - 1$, так как $Ф(-\gamma) = 1 - Ф(\gamma)$

## Оценка максимального правдоподобия
$\hat{\theta}_{ОМП}$ - оценка максимального правдоподобия
$\hat{\theta}_{ОМП}\sim^{n \to \infty} N(\theta, \frac{1}{n \cdot i(\theta)})$
$P(\gamma_1 < \sqrt{n \cdot i(\theta)}(\hat{\theta}_{ОМП} - \theta) < \gamma-2) \approx Ф(\gamma_2) - Ф(\gamma_1)$
