#матстат 
$\varkappa$ - выборочное пространство
$\varkappa = \{ \vec{x} \}, \ \mathcal{F}$ - множество допустимых распределений
$\mathcal{F}_0 \subset \mathcal{F}$ - основная гипотеза
$\mathcal{F}_1 = \mathcal{F} \setminus \mathcal{F}_0$ - альтернативная гипотеза

## Критерий проверки гипотезы
Критерием для проверки гипотезы назовём процесс проверки принадлежности наблюдаемой выборки выбранному подмножеству из $\varkappa$
- $\varkappa_0 \subset \varkappa$ - на них принимаем основную гипотезу
- $\varkappa_1 = \varkappa \setminus \varkappa_1$ - отвергаем основную гипотезу (или принимаем альтернативную)

$\varkappa_0 \subset \varkappa$ - область принятий решений
$\varkappa_1 = \varkappa \setminus \varkappa_0$ - критическая область

Критерий:
- Если $\vec{x} \in \varkappa_0$, то верна основная гипотеза
- Если $\vec{x} \in \varkappa_1$, то основная гипотеза отвергается
