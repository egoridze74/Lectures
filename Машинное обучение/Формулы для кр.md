### Линейная регрессия
$Q(w) = \sum\limits_{i = 1}^{N}(y_i - (w_0 + w_1x_i))^2$ - среднеквадратичная ошибка (MSE) без регуляризации
Среднеквадратичная ошибка с регуляризацией:
$L1: \ Q(w) = MSE + \alpha (|w_0| + |w_1|)$
$L2: \ Q(w) = MSE + \alpha (w_0^2 + w_1^2)$

### Логистическая регрессия
$\log \frac{P(y = 1 \ |x)}{P(y = -1 \ |x)} = a(x) = w_0 + w_1x$ - формула связи скалярного произведения и вероятности

### Метрики качества классификации
$Acc = \frac{TP + TN}{TP + TN + FP + FN}$
$Re = \frac{TP}{TP + FN}$
$Pr = \frac{TP}{TP + FP}$
$P = TP + FN$
$N = TN + FP$

### ROC-кривая
$TPR(t) = Re(t) = \frac{TP(t)}{TP(t) + FN(t)}$
$FPR(t) = \frac{FP(t)}{TN(t) + FP(t)}$
$TPR(t) = \int\limits_{t}^{+\infty}p_+(a)da$
$FPR(t) = \int\limits_{t}^{+\infty}p_-(a)da$

### Ядра
Исходное пространство:
$|x| = \sqrt{x_1^2 + x_2^2 + \dots + x_d^2}$
$\cos \varphi = \frac{(x, y)}{|x| \cdot |y|}$
Новое пространство:
$|x| = \sqrt{K(x, x)}$
$\cos \varphi' = \frac{K(x, y)}{K(x, x) \cdot K(y, y)}$

### Деревья решений
$H(p) = - \sum\limits_{i = 1}^{N} p_i \cdot log(p_i)$ - энтропия
$G(p) = 1 - \sum\limits_{i = 1}^{N} p_i^2$

### Разложение ошибки
$Err(x) = Noise(x) + Bias^2(x) + Variance(x)$
Или в буквах:
$E(y - f(x)^2) = Var(y | x) + [E(f(x) - f(x))]^2 + Var(f(x))$, где
- E - ошибка
- $Var(y | x)$ - неустранимая ошибка (шум)
- $[E(f(x) - f(x))]^2$ - смещение$^2$
- $Var(f(x))$ - разброс

### Бутстрап
При n выборках с возвращением из N элементов:
- $P(элемент не выбрал) = (1 - \frac{1}{N})^n$
- $E[непокрытых] = N \cdot (1 - \frac{1}{N})^N$ - ожидаемое число непокрытых элементов
- $P(не выбран ни разу) = (1 - \frac{1}{N})^{nN}$

### Просто формулы для вероятностей
- $P(A|B) = \frac{P(B | A) P(A)}{P(B)}$ - формула Байеса
- $P(B) = \sum\limits_{i} P(B | A_i)P(A_i)$ - полная вероятность
