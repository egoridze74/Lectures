## MSE
$L(a, y) = (a - y)^2$
$Q_i = a(x_i)$
$$Q(a, X) = \frac{1}{N} \sum\limits_{i = 1}^{N} L(a_i, y_i)$$

## MAE
$L(a, y) = |a - y|$
$$Q(a, X) = \frac{1}{N} \sum\limits_{i = 1}^{N} L|a_i, y_i|$$

## Ошибка Хьюбера
$$L_{\sigma}(a, y) = \begin{cases} \frac{1}{2} (y - a)^2, \ |y - a| < \sigma \\ \sigma(|y - a| - \frac{1}{2} \sigma), \ |y - a| \geq \sigma \end{cases}$$
