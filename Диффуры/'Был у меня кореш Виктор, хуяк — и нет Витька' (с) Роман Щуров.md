## Уравнение 1 (ДУ первого порядка)
$$dy - (y^2 + x^2 + 2xy + 3x + 3y + 1)dx = 0$$
$$\frac{dy}{dx} = y^2 + x^2 + 2xy + 3x + 3y + 1$$
Слегка преобразуем правую часть (выделим полный квадрат и поставим скобки)
$$\frac{dy}{dx} = (x + y)^2 + 3(x + y) + 1$$
Сделаем замену $z = x + y$, тогда $\frac{dz}{dx} = 1 + \frac{dy}{dx} \implies \frac{dy}{dx} = \frac{dz}{dx} - 1$ получим:
$$\frac{dy}{dx} = z^2 + 3z + 1$$
$$\frac{dz}{dx} = z^2 + 3z + 2$$
Воспользуемся [[Метод разделения переменных|методом разделения переменных]]
$$\frac{dz}{z^2 + 3z + 2} = dx$$
Понимаем, что знаменатель раскладывается в $(z + 1)(z + 2)$
Воспользуемся методом неопределённых коэффициентов:
$\frac{1}{(z + 1)(z + 2)} = \frac{A}{z + 1} + \frac{B}{z + 2}$
$Az + 2A + Bz + B = 1$
$\begin{cases} A + B = 0 \\ 2A + B = 1 \end{cases}$
Тогда $A = 1, \ B = -1$

В итоге получим: $$\frac{dz}{z + 1} - \frac{dz}{z + 2} = dx$$
Теперь интегрируем:
$$\int(\frac{1}{z + 1} - \frac{1}{z + 2})dz = \int dx$$
$$\ln|z + 1| - \ln|z + 2| = x + C$$
$$\ln|\frac{z + 1}{z + 2}| = x + C$$
Возвращаем начальные переменные:
$$\frac{x + y + 1}{x + y + 2} = x + C$$

## Уравнение 2
$$(e^y - x)dy - e^{-y}dx = 0$$
- $M = - e^{-y}$
- $N = e^{y} - x$
- $\frac{dM}{dy} = e^{-y}$
- $\frac{dN}{dx} = -1$
- $\frac{dM}{dy} \neq \frac{dN}{dx}$, значит, условие Эйлера не выполняется

Будем искать интегрирующий множитель $\mu(y)$:
$$F_2 = \frac{\frac{dN}{dx} - \frac{dM}{dy}}{M} = \frac{-1 - e^{-y}}{-e^{-y}} = \frac{1 + e^{-y}}{e^{-y}} = e^{y} + 1$$
Зависит только от y, тогда:
$$\mu(y) = \exp(\int F_2(y)dy)$$
$$\mu(y) = \exp(\int (e^y + 1)dy)$$
$$\mu = e^{e^y + y}$$
Умножаем начальное уравнение на $\mu$:
$$e^{e^{y} + y} \cdot (-e^{-y}dx + (e^y - x)dy) = 0$$
$$-e^{e^{y}}dx + e^{e^{y}}(e^{2y} - xe^{y})dy = 0$$
Тогда коэффициенты:
- $\tilde{M} = -e^{e^{y}}$
- $\tilde{N} = e^{e^{y}}(e^{2y} - xe^{y})$
- $\frac{d\tilde{M}}{dy} = -e^{e^{y} + y}$
- $\frac{d\tilde{N}}{dx} = -e^{e^{y} + y}$
- $\frac{dM}{dy} = \frac{dN}{dx}$ - условие Эйлера выполняется, значит ищем $U(x, y)$

Ищем функцию $U(x, y)$:
1. $\frac{dU}{dx} = \tilde{M} = -e^{e^{y}}$
	$U(x, y) = -xe^{e^{y}} + \varphi(y)$
2. $\frac{dU}{dy} = \tilde{N} = e^{e^{y}}(e^{2y} - xe^{y})$
	$\frac{dU}{dy} = (-xe^{e^{y}} + \varphi(y))' = -xe^{e^{y}} \cdot e^{y} + \varphi'(y) = -x e^{e^{y} + y} + \varphi'(y)$
3. Приравниваем и ищем $\varphi(y)$:
	$-x e^{e^{y} + y} + \varphi'(y) = -xe^{e^{y}} + e^{e^{y} + 2y} \implies \varphi'(y) = e^{e^{y} + 2y}$
	$\varphi(y) = \int e^{e^{y} + 2y}dy$ (решение этого интеграла предоставлено Раковым Богданом)
	$\int e^{2y} \cdot e^{e^y}dy = \begin{vmatrix}a = e^y \\ da = e^ydy\end{vmatrix} = \int e^a \cdot a da = \begin{vmatrix}u = a, \ du = da \\ dv = e^a da \\ v = e^a\end{vmatrix} = ae^a - \int e^a da =$
	$= ae^a - e^a + C = e^y \cdot e^{e^{y}} - e^{e^{y}} + C = e^{e^{y}}(e^{y} - 1) + C$

**Итоговый ответ:**
$$-xe^{e^{y}} + e^{e^{y}}(e^{y} - 1) = C$$
$$e^{e^{y}}(e^{y} - x - 1) = C$$

## Уравнение 3
Тут нам сказали решать через частное решение уравнения Риккати. Это одно
$$x^2dy + (3x^2y^2 - 4xy - 50)dx = 0$$
Стандартный вид уравнения Риккати из лекций:
$$\frac{dy}{dt} + p(t)y + \varphi(t)y^2 = \psi(t)$$
Приводим к этому виду наше уравнение:
$$\frac{dy}{dx} + 3y^2 - \frac{4y}{x} - \frac{50}{x^2} = 0$$
$$\frac{dy}{dx} = \frac{50}{x^2} + \frac{4y}{x} - 3y^2$$
В задании было сказано найти частное решение в виде $w(x) = \frac{a}{x}$. Подставим это в полученное уравнение:
$$-\frac{a}{x^2} = \frac{50}{x^2} + \frac{4}{x} \cdot \frac{a}{x} - 3 \cdot \frac{a^2}{x^2}$$
$$-a = 50 + 4a - 3a^2$$
$$3a^2 - 5a - 50 = 0$$
- $D = 25 + 600 = 625$
- $a_1 = 5, \ a_2 = -\frac{10}{3}$ Вроде как можно взять любой
- Тогда частное решение будет: $w(x) = \frac{5}{x}$

Знаем, что $y(x) = v(x) + w(x) = v(x) + \frac{5}{x}$
Подставляем в начальное уравнение:
$$-\frac{5}{x^2} + \frac{dv}{dx} = \frac{50}{x^2} + \frac{4}{x}(\frac{5}{x} + v) - 3(\frac{5}{x} + v)^2$$
$$- \frac{5}{x^2} + \frac{dv}{dx} = \frac{50}{x^2} + \frac{20}{x^2} + \frac{4}{x}v - \frac{75}{x^2} - \frac{30}{x}v + v^2$$
$$\frac{dv}{dx} = -\frac{26}{x}v - 3v^2$$
Теперь решаем это как уравнение Бернулли:
$$\frac{dv}{dx} + \frac{26}{x}v + 3v^2 = 0$$
$\alpha = 2$, значит делаем замену: $z = v^{1 - 2} = \frac{1}{v}$
Тогда получаем:
$$-\frac{1}{v^2} \frac{dz}{dx} + \frac{26}{xu} + \frac{3}{v^2}$$
$$\frac{dz}{dx} - \frac{26}{x}v = 3$$
Тогда:
- $p(x) = -\frac{26}{x}$
- $f(x) = 3$
- $A(x) = \int p(x)dx = -26 \int \frac{1}{x}dx = -26\ln|x| + C_1$
- $R(x) = \int f(x) \cdot e^{A(x)}dx = 3\int x^{-26}dx = -\frac{3}{25}x^{-25} + C_2$
$$z(x) = C \cdot x^{26} - \frac{3}{25}x$$
$$z(x) = \frac{1}{v} \implies v = \frac{1}{Cx^{26} - \frac{3}{25}x}$$

Тогда:
$$y(x) = v(x) + w(x) = \frac{5}{x} + \frac{1}{Cx^{26} - \frac{3}{25}x}$$

## Уравнение 4
$$\ddot{y} - 7\dot{y} + 12y = sh(3t) + sin(3t)$$
$$\ddot{y} - 7\dot{y} + 12y = \frac{e^{3t}}{2} - \frac{e^{-3t}}{2} + \sin(3t)$$
Характеристическое уравнение:
$$\lambda^2 - 7 \lambda + 12 \lambda = 0$$
Дискриминантом мозги не делаем, по Виета будет $\lambda_1 = 3, \ \lambda_2 = 4$

Корни действительные и разные, поэтому общее решение:
$$u(t) = C_1 e^{3t} + C_2 e^{4t}$$
Дальше ищем частные решения.
Правая часть у нас $\frac{e^{3t}}{2} + \frac{e^{-3t}}{2} + \sin(3t)$, частное решение ищем в виде суммы: $w(t) = w_1(t) + w_2(t) + w_3(t)$, то есть:
- $\ddot{w}_1 - 7\dot{w}_1 + 12w_1 = \frac{e^{3t}}{2}$
- $\ddot{w}_2 - 7\dot{w}_2 + 12w_2 = \frac{e^{-3t}}{2}$
- $\ddot{w}_3 - 7\dot{w}_3 + 12w_3 = \sin(3t)$

Найдём $w_1$:
$\ddot{w}_1 - 7\dot{w}_1 + 12w_1 = \frac{1}{2} e^{3t}$
3 - корень характеристического уравнения кратности $k=1$, значит решение имеет вид $A \cdot t^k \cdot e^{\alpha t}$
- $w_1 = A \cdot te^{3t}$
- $\dot{w}_1 = A(e^{3t} + 3te^{3t}) = A(1 + 3t)e^{3t}$
- $\ddot{w}_1 = A(3e^{3t} + 3e^{3t} + 9te^{3t}) = A(6 + 9t)e^{3t}$
Подставляем в уравнение:
$$A(6 + 9t)e^{3t} - 7A(1 + 3t)e^{3t} + 12Ate^{3t} = \frac{e^{3t}}{2}$$
$$e^{3t}(6A + 9At - 7A - 21At + 12At) = \frac{e^{3t}}{2}$$
$$A = - \frac{1}{2}$$
Значит, $w_1 = -\frac{te^{3t}}{2}$

Найдём $w_2$:
$\ddot{w}_2 - 7\dot{w}_2 + 12w_2 = - \frac{1}{2} e^{-3t}$
-3 - не корень характеристического уравнения, значит решение имеет вид $Ae^{\alpha t}$
- $w_2 = Ae^{-3t}$
- $\dot{w}_2 = -3Ae^{-3t}$
- $\ddot{w}_2 = 9Ae^{-3t}$
Подставляем в уравнение:
$$9Ae^{-3t} + 21Ae^{-3t} + 12Ae^{3t} = - \frac{1}{2}e^{-3t}$$
$$42A = - \frac{1}{2}$$
$$A = - \frac{1}{84}$$
Значит, $w_2 = - \frac{e^{-3t}}{84}$

Найдём $w_3$:
$\ddot{w}_3 - 7\dot{w}_3 + 12w_3 = \sin(3t)$
Правая часть $\sin(3t)$ соответствует вот этому из таблички:
$P_m(t) \cos(bt) + Q_m(t) \sin(bt)$ (b - как коэффициент мнимой части комплексного числа)
Числа $\pm 3i$ не являются корнями характеристического уравнения, значит решение имеет вид $M \cos(3t) + N \sin(3t)$
- $w_3 = M \cos(3t) + N \sin(3t)$
- $\dot{w}_3 = -3M \sin(3t) + 3N \cos(3t)$
- $\ddot{w}_3 = -9M \cos(3t) - 9N \sin(3t)$
Подставляем в уравнение:
$$-9M \cos(3t) - 9N \sin(3t) + 21M \sin(3t) - 21N \cos(3t) + 12M \cos(3t) + 12N \sin(3t) = \sin(3t)$$
$$\cos(3t)(3M - 21N) + \sin(3t)(3N + 21M - 1) = 0$$
Тогда:
- $3M = 21N \implies M = 7N$
- $3N + 21M = 1 \implies N = \frac{1}{150} \implies M = \frac{7}{150}$

Значит, $w_3 = \frac{7}{150}\cos(3t) + \frac{1}{150}\sin(3t)$

Собираем итоговое решение $y(t) = u(t) + w_1(t) + w_2(t) + w_3(t)$

Итоговое решение:
$$y(t) = C_1 e^{3t} + C_2 e^{4t} - \frac{te^{3t}}{2} - \frac{e^{-3t}}{84} + \frac{7}{150}\cos(3t) + \frac{1}{150}\sin(3t)$$
