### Идея
Ставить минимальный элемент массива в начало.

### Алгоритм
На шаге i ищем минимальное значение из элементов $a[i], \dots, a[n]$ и меняем его местами с $a[i]$

### Реализация
``` Python
arr = [...]
n = len(arr)

for i in range(n):
	k = i
	x = arr[i]
	for j in range(i + 1, n):
		if a[j] < x:
			k = j
			x = arr[j]
	a[k] = a[i]
	a[i] = x
```

### Оцениваем по критериям:
1. Алгоритмическая сложность: $O(n^2)$
2. Дополнительные затраты по памяти: $O(1)$
3. Устойчивость: нет
4. Естественность поведения: нет
