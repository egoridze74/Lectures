Выделяют 2 типа баз данных:

### ACID
- Atomicity (операция либо происходит полностью, либо вообще не совершается)
- Consistency (переход к новому состоянию не нарушает правил)
- Isolation (операции блокируют только те данные, с которыми работают)
- Durability (после коммита изменения сохраняются навсегда)

### BASE
- Basically available (есть ответ даже при сбое или недоступности)
- Soft state (состояние данных может меняться со временем)
- Eventually consistent (данные обмениваются между узлами, чтобы быть доступными везде)
