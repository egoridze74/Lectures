#рксп
Препод: Трихунков Михаил Сергеевич
Формула: $0.6 * O_{накоп} + 0.4 * O_{экз}$
$O_{накоп} = 0.1 * оформление + 0.1 * посещаемость + 0.8 * результат$
Результат = $\% \ \text{от лучшей работы}$

Автомат: какое-то сложное задание

В конце предмета хочется увидеть серверную и клиентскую часть ВПН-сервера, запущенные на контейнерах на серваке.

Что понадобится:
1. Visual Stuido Code
2. Docker
3. Dbeaver
4. Postman
5. GitLab Miem/GitHub/Gitea

На докере разворачиваем:
1. Fedora 40 (лучше его)
2. Ubuntu

Расширения для VS Code:
1. Контейнеры разработчика (Dev Containers)
2. Python
3. Yaml
4. json
5. Bash
6. SSH Open Remote

## Структура проекта
1) ДБ на SQLite
2) Серверная часть с: 
	1. Авторизацией
	2. WebViews
	3. Rest API
	4. HealthChecks
	5. Поднимем VPN (IKeV2, XRAV)
3) Клиентская часть с:
	1. Vault
	2. Pre-checks
	3. VPN Client
	4. Status Requests
4) Git Deployment + Jenkins
